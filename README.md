# EduSphere

EduSphere is a Unified Academic, Portfolio & Career Development Platform built using Django.

## Features

- Student Dashboard
- Faculty Dashboard
- HOD Dashboard
- Placement Dashboard
- Role-Based Access Control (RBAC)
- JWT Authentication
- Project Management
- Internship Tracking
- Certification Management
- Resume Generation
- Digital Portfolio
- Django Admin
- REST APIs

## Tech Stack

- Python
- Django
- Django REST Framework
- React.js
- MySQL
- Pillow
- JWT Authentication
- Bootstrap

## Installation

```bash
git clone https://github.com/yourusername/EduSphere.git
cd EduSphere
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
